How to paste into Webflow:
  1. Click the button above to copy the code
  2. In Webflow Designer, click the + button (top-left)
  3. Drag a Code Embed block onto your page
  4. Press Ctrl+V (or Cmd+V on Mac) to paste
  5. Click Save & Close
  6. Click Publish to see it live
  7. Each configurator goes on its own Webflow page!
  8. Operator Suite goes on a new /operator page

Each file is under 50,000 characters — fits in a single Webflow embed block.
The configurator only renders on the published site, not in the Designer.
